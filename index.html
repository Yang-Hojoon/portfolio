Background

그린컴퓨터아카데미에서 UI/UX 웹퍼블리셔 과정을 수강하며 기획, 디자인, 프론트엔드 구현까지 웹 개발 전반을 학습했습니다.

HTML, CSS, JavaScript, jQuery, React, PHP를 익히며 사용자 경험을 고려한 웹 서비스 제작 과정을 몸으로 배웠습니다.

Goal

사용자의 불편을 발견하고 이를 더 나은 경험으로 개선하는 UX 기획자가 목표입니다.

사용자 리서치와 데이터를 기반으로 문제를 정의하고, 사용자에게 더 나은 경험을 제공하는 일을 하고 싶습니다.
